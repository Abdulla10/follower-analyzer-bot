# 🤖 Follower Analyzer Bot

بوت تيليغرام متخصص في تحليل حسابات **Instagram** و**TikTok** وتقديم تقارير شاملة عن جودة المتابعين ومعدل التفاعل وتقييم الحساب.

---

## 📋 المميزات

| الميزة | الوصف |
|--------|-------|
| 🔍 تحليل الحساب | تحليل كامل لأي حساب عام على Instagram أو TikTok |
| 📊 معدل التفاعل | حساب دقيق بناءً على آخر 12 منشور |
| 👥 تحليل المتابعين | تقدير نسبة المتابعين الحقيقيين / غير النشطين / الوهميين |
| 📈 تحليل النمو | تحديد ما إذا كان النمو طبيعياً أو مشبوهاً |
| ⭐ التقييم النهائي | تقييم شامل من 100 نقطة مع مستوى (ممتاز / جيد / متوسط / ضعيف) |
| 🔄 مقارنة حسابين | مقارنة تفصيلية بين حسابين مع تحديد الفائز |

---

## 🚀 خطوات التشغيل

### 1. إنشاء البوت على تيليغرام

1. افتح تيليغرام وابحث عن **@BotFather**
2. أرسل الأمر `/newbot`
3. اختر اسماً للبوت: `Follower Analyzer Bot`
4. اختر username للبوت: مثلاً `follower_analyzer_bot`
5. **انسخ التوكن** الذي يعطيك إياه BotFather

### 2. تثبيت المتطلبات

```bash
pip install -r requirements.txt
```

### 3. تعيين التوكن

**الطريقة الأولى (موصى بها) — متغير البيئة:**
```bash
export BOT_TOKEN="1234567890:ABCdefGHIjklMNOpqrSTUvwxYZ"
```

**الطريقة الثانية — تعديل الكود مباشرة:**

افتح ملف `bot.py` وعدّل السطر:
```python
BOT_TOKEN = os.environ.get("BOT_TOKEN", "YOUR_BOT_TOKEN_HERE")
```
إلى:
```python
BOT_TOKEN = "1234567890:ABCdefGHIjklMNOpqrSTUvwxYZ"
```

### 4. تشغيل البوت

```bash
python bot.py
```

ستظهر الرسالة:
```
🤖 Follower Analyzer Bot يعمل الآن...
```

## 🔧 طريقة عمل البوت

### تدفق المحادثة

```
المستخدم يضغط /start
        ↓
القائمة الرئيسية (4 أزرار)
        ↓
[تحليل حساب] → اختيار المنصة → إرسال اليوزر → التقرير
[مقارنة حسابين] → حساب 1 → حساب 2 → تقرير المقارنة
[اشتراك VIP] → عرض مميزات VIP
[المساعدة] → دليل الاستخدام
```

### معادلة التفاعل

```
معدل التفاعل = (متوسط الإعجابات + متوسط التعليقات) ÷ عدد المتابعين × 100
```

### تصنيف معدل التفاعل

| النسبة | التصنيف |
|--------|---------|
| 6% فأكثر | ممتاز 🔥 |
| 3% - 6% | جيد ✅ |
| 1% - 3% | متوسط ⚠️ |
| أقل من 1% | ضعيف ❌ |

### نظام التقييم (100 نقطة)

| المعيار | الوزن |
|---------|-------|
| معدل التفاعل | 50 نقطة |
| جودة المتابعين | 30 نقطة |
| طبيعة النمو | 20 نقطة |

## ⚙️ ملاحظات تقنية

- **البيانات الحية:** يحاول البوت جلب البيانات الفعلية من Instagram وTikTok عبر واجهات الويب العامة.
- **البيانات التقديرية:** في حال عدم توفر البيانات (حساب خاص أو قيود API)، يستخدم البوت خوارزمية تقدير واقعية ويُشير إلى ذلك في التقرير.
- **تحليل المتابعين:** نسب المتابعين الحقيقيين/الوهميين هي تقديرات مبنية على معدل التفاعل ونسبة المتابعة، وليست بيانات دقيقة 100%.

## 🛠️ التخصيص

### تغيير اسم الدعم الفني (VIP)

في ملف `config.py`:
```python
SUPPORT_USERNAME = "@YourSupportUsername"
```

### تغيير عدد المنشورات المحللة

في ملف `config.py`:
```python
FREE_POSTS_LIMIT = 12   # للمستخدمين المجانيين
VIP_POSTS_LIMIT = 50    # لمستخدمي VIP
```
